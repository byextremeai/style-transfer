# Visual Style Analysis Reference

Complete reference for analyzing the visual style of images.
All keywords are ready to use directly in fal.ai prompts.

## 1. COLOR SYSTEM

### 1.1 Color Palette
Extract 3–5 dominant colors and estimate their HEX codes.

**Roles:**
- **Dominant** — color occupying 50%+ of the frame
- **Secondary** — second most prominent color
- **Accent** — highlight or emphasis color
- **Neutral** — background or whitespace tone

### 1.2 Color Temperature
| Category | Keywords | Character |
|----------|----------|-----------|
| Warm | warm tones, golden, amber | reds / oranges / yellows |
| Neutral | neutral tones, balanced | grays / beiges / ivories |
| Cool | cool tones, icy, steel | blues / teals / purples |

## 9. FAL.AI PROMPT ASSEMBLY GUIDE

Assemble the final prompt in this order after analysis.

### Generate Prompt (text → image)
```
[subject + action], [setting/environment],
[visual style + medium], [lighting quality + source],
[color palette + grading], [composition + framing],
[texture + surface], [mood + atmosphere],
[brand personality adjectives]
```

**Example:**
```
young woman wearing minimalist sneakers walking through Tokyo streets,
urban environment with concrete and glass architecture,
editorial photography, cinematic style,
soft natural daylight, overcast diffused light,
muted earth tones with subtle warm accents, film grain, matte finish,
rule of thirds, medium shot, shallow depth of field,
smooth surfaces, clean lines,
calm yet confident, urban, understated luxury
```

### Edit Prompt (image → image)
```
transform into [visual style], [lighting], [color grading],
[mood], maintaining the product shape and details,
[key brand personality]
```

**Example:**
```
transform into editorial lifestyle photo, soft natural backlight,
warm muted color grading with film grain, calm and confident mood,
maintaining the product shape and details exactly,
urban minimalist aesthetic
```
